# greattree

> *"The Erdtree governs all. The Erdtree blesses all. The Erdtree connects all."*

A custom Software Defined Networking (SDN) stack built from scratch in C++. Part of the [wasoulcloud](https://github.com/1wolfalone1/wasoulcloud) ecosystem.

Like the Erdtree whose roots spread beneath the Lands Between connecting every region invisibly — greattree is the network fabric that connects every VM across every node. We wanted `waerdtree`. It was taken. The Tarnished adapts.

## What is this?

An SDN built the hard way. No pre-built virtual switches. Raw sockets, packet parsing, tunneling, firewalling — all written from scratch in C++20. Because we like souls games, and apparently we like suffering in code too.
